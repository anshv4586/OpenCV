# # OpenCV Project 👁️‍🗨️

This repository contains computer vision projects and experiments using **OpenCV (Open Source Computer Vision Library)**. It demonstrates image processing, object detection, face recognition, and other vision-based applications.

---

## 📌 Features

- Image reading, writing, and displaying
- Basic image processing (blur, edge detection, thresholding)
- Face detection using Haar Cascades / DNN
- Object detection
- Video capture and processing
- Real-time webcam applications
- Contour detection and shape recognition

---

## 🛠️ Technologies Used

- Python 🐍
- OpenCV (`cv2`)
- NumPy
- Matplotlib (optional for visualization)

---

## 📦 Installation

```bash

pip install numpy
pip install matplotlib
